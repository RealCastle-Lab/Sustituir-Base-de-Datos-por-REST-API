Acerca de la Aplicación

Esta aplicación fue desarrollada para ofrecer una interfaz web fácil de usar que permita a los usuarios consultar datos de vacunación contra el sarampión en niños entre 12-23 meses en Panamá.

Características:

  • Consulta de datos históricos de vacunación.
  • Visualización de datos a través de una interfaz web interactiva.
